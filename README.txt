# Rino Windows Security Audit Toolkit

PowerShell toolkit for performing automated security audits on Windows systems.
Generates detailed security reports with key system information.

---

## Description

Windows Security Audit Toolkit is a lightweight PowerShell tool designed for system administrators and technical support professionals.

It performs automated security checks and generates detailed reports to help identify potential risks and misconfigurations.

---

## Features

- Firewall status verification
- Detection of open listening ports
- Administrator users listing
- Windows Update service status
- Remote Desktop (RDP) configuration
- Antivirus / Windows Defender status
- Shared folders detection
- Recent security event logs
- Automatic report generation
